# 🍓 Fresh Fruits  Website

## 📌 Description
This is a simple multi-page website created using only HTML.  
The website provides information about different fruits, their prices, and health benefits.

## 🚀 Features
- Multi-page website (Home, Fruits, Benefits, Contact)
- Simple and clean design
- Easy navigation using links
- Images added for better understanding
- Beginner-friendly project

## 🛠️ Technologies Used
- HTML

## 📂 Project Structure
fruits-website/
│
├── index.html
├── fruits.html
├── benefits.html
├── contact.html
│
└── images/
    ├── apple.jpg
    ├── banana.jpg
    ├── mango.jpg

## ▶️ How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Double click on `index.html`
4. The website will open in your browser

## 🌐 Pages Included
- Home Page (index.html)
- Fruits Page (fruits.html)
- Benefits Page (benefits.html)
- Contact Page (contact.html)
